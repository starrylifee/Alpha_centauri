# Proxima Rescue Command (PRC)

2026 영재 위탁 프로그램 '프록시마 센타우리' 수업 보조용 웹앱

## 프로젝트 개요

우주선 관제 센터 인터페이스를 모티브로 한 방탈출 게임 형식의 수업 진행 도구입니다.

## 실행 방법

1. `index.html` 파일을 웹 브라우저(Chrome 권장)로 열기
2. 별도의 서버 설정 없이 바로 실행 가능

## 주요 기능

- 5단계 선형 진행 시스템
- LocalStorage 기반 진행 상황 저장
- 실시간 타이머
- 힌트 시스템
- 정답 검증 시스템 (해시 기반 보안)

## 프로젝트 구조

자세한 내용은 `PRD.md` 파일을 참조하세요.

## 개발 상태

현재 프로젝트 구조 및 PRD 문서 작성 완료. 구현 단계 진행 예정.
